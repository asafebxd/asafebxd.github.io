Asafe

Youtube clone

Este projeto é uma recriação do YouTube utilizando HTML, CSS e JavaScript, proporcionando uma experiência de navegação e visualização de vídeos similar à plataforma original. A interface inclui uma barra lateral com opções de navegação, uma área de pesquisa e um feed dinâmico de vídeos recomendados. Ele foi desenvolvido de forma responsiva, adaptando-se a diferentes tamanhos de tela, com um design agradável e intuitivo, utilizando ícones do Google Material Design para uma melhor interação do usuário
