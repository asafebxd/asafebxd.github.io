Spotify Clone

Este projeto é um clone do Spotify, criado com HTML, CSS e JavaScript. Ele simula uma interface de música, com barra lateral de navegação, seção de playlists, busca e filtros. O layout é interativo e responsivo, proporcionando uma experiência de usuário próxima à do Spotify real. Ideal para quem quer aprender sobre criação de interfaces dinâmicas e design.
